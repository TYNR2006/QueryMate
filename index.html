Relational Database Model

The relational model organizes data into tables with rows and columns. Each table represents an entity, and relationships between entities are established through foreign keys.

What is the primary purpose of normalization in database design?

Explanation

Normalization is a database design technique that reduces data redundancy and improves data integrity by organizing fields and tables in a database. It involves dividing large tables into smaller ones and defining relationships between them.
